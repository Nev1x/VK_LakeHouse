# Intent-brief 004 — gold-marts

**Цель.** Gold-витрины поверх silver: агрегаты для аналитики/дашбордов + feature-таблица для
будущего ML («лофт / не лофт»).

**Scope.**
- Витрины-агрегаты: распределение цен/площадей по районам, динамика загрузок, срезы по
  стилю/ремонту/мебели (финальный список — в спеке).
- Feature-таблица `apartments_features`: нормализованные признаки квартиры + ссылки на фото —
  вход для датасета обучения (006).
- Схемы gold — замороженный контракт (устав I-6): изменения только additive/versioned.
- Тот же механизм Trino SQL + Python-раннер, инкрементальные пересчёты.

**Вне scope.** Сами дашборды (005), экспорт датасета (006), обучение модели (вне платформы).

**Зависимости.** 003.

**Готово, когда.** Витрины считаются из silver воспроизводимо; схемы зафиксированы и
задокументированы; SELECT'ы для дашбордов укладываются в разумное время на демо-объёме.
